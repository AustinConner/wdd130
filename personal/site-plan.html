My Projects

At work, we have a lot of computers that need to be deployed and not a streamlined process to get them done. We have to go through each device manually to get it up and running. After some research, I learned that I can use windows deployment server and windows development toolkit to PXE boot a machine and pretty much automate everything. I haven’t made it too far yet through, I’m hung up creating an Active Directory server so I can have it as close to the environment we have at work without actually having it. The idea is that we plug the device into the network and boot it off of it, then we can give the computer a name so we can find it in our RMM. Once the device is named, it’ll join the domain and install a few essential applications to inferface with the rmm. Once we have it on the domain and connected to the rmm, we can use the rmm to install the rest of the applications silently. Hopefully cutting down the deployment time for devices from 1 hour to just 2 minutes or so.
